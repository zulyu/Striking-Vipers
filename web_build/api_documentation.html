Striking Vipers API Documentation

API version: 1.0

This documentation provides information about the available endpoints in the Striking Vipers API.

Authentication

POST /api/auth/login

Authenticate a user and return a JWT token.

Request Body

Parameter Type Required Description
email string Yes User's email address
password string Yes User's password

Responses

  • 200 Login successful
  • 401 Invalid credentials
POST /api/auth/signup

Register a new user (student or teacher).

Request Body

Parameter Type Required Description
name string Yes User's full name
email string Yes User's email address
password string Yes User's password
role string Yes User's role ("student" or "teacher")
class_code string No (Yes for students) Class code (required for students)

Responses

  • 201 User created successfully
  • 400 Invalid input

Teachers

GET /api/teachers

Get a list of all teachers.

Responses

  • 200 List of teachers returned successfully
POST /api/teachers

Create a new teacher.

Request Body

Parameter Type Required Description
name string Yes Teacher's name
email string Yes Teacher's email
subject string Yes Subject taught by the teacher

Responses

  • 201 Teacher created successfully
  • 400 Invalid input / Email already exists
GET /api/teachers/{teacher_id}

Get a specific teacher by ID.

Path Parameters

Parameter Type Required Description
teacher_id integer Yes Teacher's unique identifier

Responses

  • 200 Teacher returned successfully
  • 404 Teacher not found
PUT /api/teachers/{teacher_id}

Update a specific teacher.

Path Parameters

Parameter Type Required Description
teacher_id integer Yes Teacher's unique identifier

Request Body

Parameter Type Required Description
name string Yes Teacher's name
email string Yes Teacher's email
subject string Yes Subject taught by the teacher

Responses

  • 200 Teacher updated successfully
  • 400 Invalid input / Email already exists
  • 404 Teacher not found
DELETE /api/teachers/{teacher_id}

Delete a specific teacher.

Path Parameters

Parameter Type Required Description
teacher_id integer Yes Teacher's unique identifier

Responses

  • 204 Teacher deleted successfully
  • 404 Teacher not found
